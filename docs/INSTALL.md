# Installation dans ton Laravel 8

1. Dézipper ce pack.
2. Copier tous les dossiers/fichiers dans le projet Laravel 8.
3. Accepter le remplacement de `routes/web.php` et `database/seeders/DatabaseSeeder.php`.

Commandes :

```bash
php artisan migrate
php artisan db:seed
php artisan serve --host=0.0.0.0
```

URLs : `/admin`, `/blog`, `/offres`, `/realisations`, `/sitemap.xml`, `/sitemap-blog.xml`, `/robots.txt`
